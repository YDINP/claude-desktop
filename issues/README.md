# Issues

이슈 발견 시 이 폴더 루트에 MD 파일을 직접 추가하세요.
처리 완료된 파일은 `Done/` 폴더로 자동 이동됩니다.

## 폴더 구조
```
issues/
├── ISSUE-001-버그명.md     ← 미처리 이슈
├── ISSUE-002-기능요청.md
└── Done/
    ├── ISSUE-000-예시.md   ← 완료된 이슈
    └── ...
```

## 파일명 규칙
```
ISSUE-NNN-간단한-제목.md
```

## 파일 형식 (권장)
```markdown
# ISSUE-NNN — 제목

**유형**: bug | feature | ux | performance
**우선순위**: high | medium | low
**관련 파일**: src/...

## 현상
...

## 기대 동작
...

## 메모
...
```

---
루프는 각 라운드 완료 후 루트 이슈 파일을 스캔하고,
처리 완료 시 해당 파일을 `Done/` 폴더로 이동합니다.
