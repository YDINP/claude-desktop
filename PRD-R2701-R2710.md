## R2701 — SceneView 마르키 선택 (Rubber-band Select)

**파일:** CCFileSceneView.tsx
**설명:** 빈 공간 mousedown → 파란 반투명 사각형 드래그 → mouseup 시 사각 내 노드 multiSelect
**State:** `marqueeStart: {x,y}|null`, `marqueeEnd: {x,y}|null`
**로직:**
- 배경 rect mousedown (노드 위 아닌 경우) → marqueeStart 설정
- mousemove → marqueeEnd 업데이트, SVG에 파란 rect 렌더
- mouseup → flatNodes 중 AABB overlap 노드 → setMultiSelected (Shift키면 기존 유지, 없으면 교체)
- marquee 사각형: `fill="rgba(59,130,246,0.15)" stroke="rgba(59,130,246,0.6)" strokeWidth={1/zoom} strokeDasharray="3,2"`

## R2703 — SceneView 선택 노드 뷰 중심 팬 (Center on Selection)

**파일:** CCFileSceneView.tsx
**설명:** 툴바 ⊕C 버튼 클릭 시 선택 노드(들) 평균 중심점으로 뷰 팬
**로직:**
- selectedUuid 있으면 해당 노드 svgPos 계산
- multiSelected 있으면 모든 노드 svgPos 평균 계산
- `setView(v => ({ ...v, panX: cx - containerW/2/v.zoom, panY: cy - containerH/2/v.zoom }))` 방식
- 선택 없으면 no-op

## R2704 — BatchInspector 색상 채널 오프셋 (ΔR/ΔG/ΔB)

**파일:** CocosPanel.tsx
**설명:** 선택 노드 color에 ΔR/ΔG/ΔB 를 더해 밝기/색조 일괄 조정
**State:** `colorDeltaR/G/B: number` (기본 0, 범위 -255~255)
**로직:** `applyColorDelta()` → `r = clamp(0, n.color.r + dR, 255)` (r/g/b 각각)
**UI:** ΔR/ΔG/ΔB 입력 3개 + 프리셋 버튼 (+30 밝게 / -30 어둡게)

## R2705 — SceneView Alt+drag 노드 복제

**파일:** CCFileSceneView.tsx
**설명:** 선택 노드 드래그 시 Alt 키 누르면 복제(deepCopy) 후 이동
**로직:**
- mousedown에서 `altKey` 감지 → `altDragRef.current = true`
- mouseup 시 altDragRef.current이면: deepCopyNodeWithNewUuids(선택 노드) → sibling에 삽입 → saveScene
- 원본은 그대로 유지, 복제 노드는 dragDelta만큼 offset 적용
- 복제 후 `setSelectedUuid(newUuid)` 로 포커스 이동

## R2707 — SceneView 최근 선택 노드 히스토리

**파일:** CCFileSceneView.tsx

## R2710 — BatchInspector 크기 고정값 일괄 설정

**파일:** CocosPanel.tsx
**설명:** W/H 각각 고정값 입력 후 선택 노드에 일괄 적용
**State:** `fixedSizeW: number` (0=미적용), `fixedSizeH: number` (0=미적용)
**로직:** `applyFixedSize()`:
- fixedSizeW > 0 → `n.size.width = fixedSizeW` (+ UITransform 동기화 있으면)
- fixedSizeH > 0 → `n.size.height = fixedSizeH`
- ✓W / ✓H 체크박스로 각각 활성화
**주의:** R2485(균등화)/R2595/R2634(첫째기준 통일)과 다름 — 임의 숫자 직접 지정
